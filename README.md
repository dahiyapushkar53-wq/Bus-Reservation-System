# Bus Reservation System (C)

## Introduction
This project is a console-based Bus Reservation System developed using the C programming language.  
It allows users to log in and perform basic reservation operations such as booking tickets, canceling tickets, and checking bus availability.

The project is intended for academic learning and demonstrates fundamental C programming concepts.

---

## Problem Statement
Manual bus ticket booking systems are time-consuming and prone to errors.  
This project aims to create a simple automated system that manages bus seat reservations efficiently.

---

## Project Objectives
- To design a menu-driven application in C
- To implement user authentication
- To manage bus seat booking and cancellation
- To display real-time bus status

---

## Features
- User login system
- Book bus tickets
- Cancel booked tickets
- Check bus details and available seats
- Logout option
- Simple and user-friendly interface

---

## Technologies Used
- Programming Language: C
- Compiler: GCC
- Platform: Console-based application

---

## System Requirements
- Any operating system (Windows/Linux/macOS)
- C compiler (GCC recommended)
- Basic knowledge of C programming
